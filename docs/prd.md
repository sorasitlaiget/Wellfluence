<b>1. Core Concept</b>
<i>วิสัยทัศน์และปัญหาที่แก้ไข</i>
Wellfluence คือแพลตฟอร์มการตลาดแบบอินฟลูเอนเซอร์ B2B2C ที่มุ่งเน้นในอุตสาหกรรมสุขภาพและเวลเนสโดยเฉพาะ เราเชื่อมโยงแบรนด์สินค้าและบริการด้านสุขภาพเข้ากับอินฟลูเอนเซอร์ที่มีความน่าเชื่อถือ และช่วยให้ผู้ติดตามของอินฟลูเอนเซอร์เข้าถึงผลิตภัณฑ์และบริการที่ตรงกับความต้องการและไว้วางใจได้ ปัญหาหลักที่เราแก้ไขคือแบรนด์พบความท้าทายในการค้นหาอินฟลูเอนเซอร์ที่เหมาะสม มีส่วนร่วมจริง และวัดผลได้ ในขณะที่อินฟลูเอนเซอร์ก็ขาดเครื่องมือในการจัดการความร่วมมือและสร้างรายได้อย่างมืออาชีพ แพลตฟอร์มของเราจะช่วยสร้างความโปร่งใส ความน่าเชื่อถือ และประสิทธิภาพในการทำการตลาดผ่านอินฟลูเอนเซอร์ในอุตสาหกรรมสุขภาพและเวลเนส

<b>2. Business & User Niche</b>
<i>กลุ่มลูกค้าและ Pain Points</i>
•   <b>แบรนด์ (B2B):</b> บริษัทผลิตอาหารเพื่อสุขภาพ, อุปกรณ์ออกกำลังกาย, อาหารเสริม, ผลิตภัณฑ์ดูแลผิว, ผู้ให้บริการด้านสุขภาพและเวลเนส ที่ต้องการช่องทางการตลาดแบบตรงเป้าหมายและเข้าถึงกลุ่มผู้บริโภคที่แท้จริง
    •   Pain Points: ค้นหาอินฟลูเอนเซอร์ที่ตรงกลุ่มเป้าหมายได้ยาก, ตรวจสอบความน่าเชื่อถือและประสิทธิภาพได้ลำบาก, การจัดการแคมเปญหลายอันพร้อมกัน, การวัดผลตอบแทนการลงทุน (ROI) และค่าธรรมเนียมเอเจนซี่ที่สูง
•   <b>อินฟลูเอนเซอร์ (B2C สำหรับแพลตฟอร์มเรา, B2B สำหรับแบรนด์):</b> เทรนเนอร์ฟิตเนส, นักโภชนาการ, ครูโยคะ, โค้ชด้านสุขภาพจิต, ผู้สนับสนุนการใช้ชีวิตเพื่อสุขภาพที่มีผู้ติดตามอยู่แล้ว
    •   Pain Points: ขาดการเชื่อมต่อโดยตรงกับแบรนด์ที่มีคุณภาพ, รายได้ที่ไม่สม่ำเสมอ, การจัดการความร่วมมือหลายแคมเปญ, การสร้างเนื้อหา, และการจัดทำรายงานที่เป็นมืออาชีพ
•   <b>ผู้ติดตาม/ลูกค้า (B2C - ผู้ใช้งานปลายทาง):</b> บุคคลที่สนใจสุขภาพ, การออกกำลังกาย, โภชนาการ, สุขภาพจิต ที่กำลังมองหาผลิตภัณฑ์/บริการที่น่าเชื่อถือและได้รับการแนะนำจากผู้เชี่ยวชาญ

<b>3. Revenue Model</b>
<i>รายได้และกระแสเงินสด</i>
•   <b>ค่าคอมมิชชั่น:</b> เก็บค่าธรรมเนียมเป็นเปอร์เซ็นต์จากมูลค่าแคมเปญที่ประสบความสำเร็จระหว่างแบรนด์และอินฟลูเอนเซอร์ผ่านแพลตฟอร์ม
•   <b>แพ็คเกจสมาชิกสำหรับแบรนด์:</b> นำเสนอคุณสมบัติพรีเมียม (เช่น การเข้าถึงข้อมูลเชิงลึกของอินฟลูเอนเซอร์ขั้นสูง, การวิเคราะห์แคมเปญละเอียด), การจับคู่อินฟลูเอนเซอร์ที่ได้รับการจัดลำดับความสำคัญ, และการสนับสนุนเฉพาะ
•   <b>แพ็คเกจสมาชิกสำหรับอินฟลูเอนเซอร์:</b> เพิ่มการมองเห็นโปรไฟล์, เข้าถึงข้อมูลเชิงลึกขั้นสูงเกี่ยวกับผู้ชมและประสิทธิภาพ, เครื่องมือระดับมืออาชีพสำหรับการจัดการแคมเปญ, การเข้าถึงแคมเปญของแบรนด์ก่อนใคร
•   <b>เครื่องมือ/เทมเพลตเนื้อหาที่ได้รับการสนับสนุน:</b> การเสนอเครื่องมือพรีเมียมสำหรับการสร้างเนื้อหาที่มีคุณภาพและเป็นไปตามข้อกำหนดของแบรนด์
•   <b>การบูรณาการ Affiliate Marketing:</b> ตัวเลือกสำหรับอินฟลูเอนเซอร์ในการรับค่าคอมมิชชั่นจากการขายที่เกิดขึ้นโดยตรงผ่านลิงก์เฉพาะของตนบนแพลตฟอร์ม (โดยแบรนด์เป็นผู้จัดการข้อเสนอ)

<b>4. Platform & Tech Stack</b>
<i>สถาปัตยกรรมและเทคโนโลยี</i>
•   Frontend: React.js (พร้อม Next.js สำหรับ Server-Side Rendering (SSR) และ SEO ที่ดีขึ้น)
•   Backend: Node.js (NestJS Framework สำหรับสถาปัตยกรรมที่ปรับขนาดได้และมีประสิทธิภาพ)
•   Database: PostgreSQL (สำหรับข้อมูลเชิงสัมพันธ์ที่ซับซ้อน การจัดการบัญชีผู้ใช้ แคมเปญ และธุรกรรมที่น่าเชื่อถือ) และ Redis (สำหรับ Caching และการจัดการเซสชันเพื่อประสิทธิภาพที่รวดเร็ว)
•   Infrastructure: AWS (Amazon Web Services สำหรับการโฮสต์ (EC2), การจัดเก็บไฟล์ (S3), ฐานข้อมูล (RDS), Serverless Functions (Lambda)), Cloudflare (สำหรับ CDN และความปลอดภัย), Docker & Kubernetes (สำหรับการปรับขนาดและการจัดการ Deployment)
•   Third-party APIs: Omise/Stripe (Payment Gateway สำหรับการทำธุรกรรมที่ปลอดภัย), Instagram API, TikTok API, YouTube Data API, Facebook Graph API (สำหรับการดึงข้อมูลอินฟลูเอนเซอร์ เช่น จำนวนผู้ติดตามและอัตราการมีส่วนร่วม), SendGrid (Email Service สำหรับการสื่อสาร)

<b>5.1 Authentication & User Management</b>
•   การลงทะเบียนผู้ใช้ (สำหรับแบรนด์และอินฟลูเอนเซอร์) ผ่านอีเมล/รหัสผ่าน หรือ Social Login (Google, Facebook)
•   การเข้าสู่ระบบ/ออกจากระบบที่ปลอดภัย
•   ฟังก์ชันลืมรหัสผ่าน/รีเซ็ตรหัสผ่าน
•   การยืนยันอีเมลเมื่อลงทะเบียน
•   การจัดการโปรไฟล์ (แก้ไขข้อมูลส่วนตัว, รูปโปรไฟล์, ข้อมูลติดต่อ, เปลี่ยนรหัสผ่าน)
•   การควบคุมการเข้าถึงตามบทบาท (แบรนด์, อินฟลูเอนเซอร์, ผู้ดูแลระบบแพลตฟอร์ม)
•   การปิดใช้งาน/ลบบัญชีผู้ใช้

<b>5.2 Brand Profile & Campaign Management</b>
•   การสร้างโปรไฟล์แบรนด์: ระบุข้อมูลบริษัท, อุตสาหกรรม, แนวทางของแบรนด์, กลุ่มเป้าหมาย, โลโก้, และข้อมูลติดต่อ
•   ตัวช่วยสร้างแคมเปญ (Campaign Creation Wizard) แบบมีขั้นตอน:
    •   ระบุชื่อแคมเปญ, คำอธิบาย, วัตถุประสงค์ (เช่น การรับรู้แบรนด์, ยอดขาย, การสร้าง Lead)
    •   กำหนดข้อมูลประชากรของกลุ่มเป้าหมาย (อายุ, เพศ, ความสนใจ, สถานที่)
    •   กำหนดเกณฑ์อินฟลูเอนเซอร์ที่ต้องการ (เฉพาะด้าน, ช่วงจำนวนผู้ติดตาม, ช่วงอัตราการมีส่วนร่วม)
    •   กำหนดงบประมาณแคมเปญ (ค่าธรรมเนียมคงที่, การให้ผลิตภัณฑ์ตัวอย่าง, คอมมิชชั่นจากการขาย, หรือแบบผสมผสาน)
    •   ระบุสิ่งที่อินฟลูเอนเซอร์ต้องส่งมอบ (จำนวนโพสต์, สตอรี่, Reels, ความยาววิดีโอ, แพลตฟอร์มโซเชียลมีเดียที่ต้องการ)
    •   ระบุดัชนีชี้วัดผลงานหลัก (KPIs) และตัวชี้วัดการติดตามที่ชัดเจน
    •   กำหนดช่วงเวลา: ระยะเวลาการรับสมัคร, กำหนดส่งเนื้อหา, วันที่เผยแพร่เนื้อหา
•   การค้นหาและจับคู่อินฟลูเอนเซอร์:
    •   เครื่องมือค้นหาและกรองอินฟลูเอนเซอร์ตามเฉพาะด้าน, ข้อมูลประชากรของผู้ชม, จำนวนผู้ติดตาม, อัตราการมีส่วนร่วม, และผลงานแคมเปญที่ผ่านมา
    •   ระบบการจับคู่ด้วย AI ที่แนะนำอินฟลูเอนเซอร์ตามเกณฑ์แคมเปญและโปรไฟล์อินฟลูเอนเซอร์
•   การเชิญอินฟลูเอนเซอร์และการจัดการใบสมัคร:
    •   ส่งคำเชิญโดยตรงไปยังอินฟลูเอนเซอร์ที่สนใจ
    •   ตรวจสอบใบสมัครของอินฟลูเอนเซอร์ที่เข้ามา (ดูพอร์ตโฟลิโอ, ตัวชี้วัดการมีส่วนร่วมที่ดึงมาจาก API)
    •   ยอมรับ/ปฏิเสธใบสมัครพร้อมข้อความแจ้ง
•   การอนุมัติบรีฟแคมเปญและเนื้อหา:
    •   แบ่งปันบรีฟแคมเปญฉบับละเอียดกับอินฟลูเอนเซอร์ที่ได้รับเลือกผ่านแพลตฟอร์ม
    •   อินฟลูเอนเซอร์ส่งฉบับร่างเนื้อหาสำหรับการตรวจสอบโดยแบรนด์
    •   แบรนด์ให้ข้อเสนอแนะและอนุมัติ/ขอแก้ไขเนื้อหา
•   การติดตามผลและรายงานประสิทธิภาพแคมเปญ:
    •   แดชบอร์ดแสดงสถานะแคมเปญแบบเรียลไทม์ (การเผยแพร่, การมีส่วนร่วม)
    •   ตัวชี้วัดประสิทธิภาพ: Reach, Impressions, Engagement Rate, Clicks, Conversions (หากมีการติดตามผ่านลิงก์เฉพาะ)
    •   ความสามารถในการสร้างรายงานที่กำหนดเองและดาวน์โหลดได้
•   การจัดการงบประมาณและการชำระเงิน:
    •   ระบบ Escrow เพื่อเก็บเงินค่าแคมเปญอย่างปลอดภัยจนกว่าแคมเปญจะเสร็จสมบูรณ์/ได้รับการอนุมัติ
    •   การปล่อยเงินชำระไปยังอินฟลูเอนเซอร์โดยอัตโนมัติเมื่อเงื่อนไขแคมเปญเป็นไปตามข้อกำหนด
    •   การออกใบแจ้งหนี้และใบเสร็จรับเงินสำหรับแบรนด์

<b>5.3 Influencer Profile & Opportunity Management</b>
•   การสร้างโปรไฟล์อินฟลูเอนเซอร์:
    •   ประวัติส่วนตัว, ความเชี่ยวชาญ, เฉพาะด้าน (Niche), ข้อมูลประชากรของผู้ชม
    •   เชื่อมต่อบัญชีโซเชียลมีเดีย (Instagram, TikTok, YouTube, Facebook) เพื่อดึงข้อมูลอัตโนมัติเกี่ยวกับผู้ติดตามและประสิทธิภาพ
    •   แสดงผลงาน/พอร์ตโฟลิโอแคมเปญที่ผ่านมา
    •   การกำหนดบริการและราคา (เช่น ค่าธรรมเนียมคงที่ต่อโพสต์, สตอรี่, วิดีโอ, หรือราคาแพ็คเกจ)
    •   การตั้งค่าข้อมูลการชำระเงินสำหรับการรับเงิน
•   การค้นหาและสมัครแคมเปญ:
    •   เรียกดูแคมเปญที่มีอยู่บนแพลตฟอร์ม (กรองตามเฉพาะด้าน, ค่าตอบแทน, สิ่งที่ส่งมอบ)
    •   สมัครแคมเปญพร้อมข้อความ/ข้อเสนอที่กำหนดเองเพื่อเสนอตัวกับแบรนด์
•   แดชบอร์ดความร่วมมือ:
    •   ดูแคมเปญที่กำลังดำเนินการ, รอดำเนินการ, เสร็จสมบูรณ์ทั้งหมด
    •   รับบรีฟแคมเปญและสื่อสารกับแบรนด์ผ่านระบบข้อความในแพลตฟอร์ม
    •   ส่งฉบับร่างเนื้อหาเพื่อขออนุมัติจากแบรนด์
    •   ติดตามสถานะการชำระเงินและดูประวัติการทำธุรกรรม
•   การวิเคราะห์และข้อมูลเชิงลึก:
    •   แดชบอร์ดแสดงผลงานโซเชียลมีเดียโดยรวม (การเติบโตของผู้ติดตาม, อัตราการมีส่วนร่วม, Reach)
    •   ตัวชี้วัดประสิทธิภาพเฉพาะแคมเปญที่เข้าร่วม
•   เครื่องมือการสร้างรายได้:
    •   การสร้างลิงก์ Affiliate สำหรับผลิตภัณฑ์/แบรนด์เฉพาะที่ร่วมรายการ
    •   ตัวเลือกในการสร้างและขายผลิตภัณฑ์ดิจิทัลของตนเอง (eBooks, คอร์สออนไลน์ - อาจเป็นขอบเขตในอนาคต)
•   การส่งข้อความโดยตรงกับแบรนด์สำหรับข้อสงสัยหรือการประสานงาน

<b>5.4 Customer/Follower Engagement (Indirect via Influencers)</b>
•   ไม่มีบัญชีผู้ใช้แพลตฟอร์ม Wellfluence โดยตรงสำหรับผู้ติดตาม
•   การค้นพบผลิตภัณฑ์และบริการผ่านเนื้อหาอินฟลูเอนเซอร์ที่แชร์บนโซเชียลมีเดียของพวกเขา
•   ลิงก์โดยตรงจากเนื้อหาอินฟลูเอนเซอร์ไปยังหน้าผลิตภัณฑ์/บริการของแบรนด์ (ผ่านลิงก์ติดตามที่กำหนดโดย Wellfluence)
•   หน้า Landing Page เฉพาะอินฟลูเอนเซอร์สำหรับการโปรโมทผลิตภัณฑ์ (จัดการโดยแบรนด์, ติดตามประสิทธิภาพโดย Wellfluence)

<b>📄 Product Requirements Document</b>
<b>Project:</b> Wellfluence
<b>Version:</b> 1.0

<b>1. Core Concept</b>
Wellfluence is a B2B2C influencer marketing platform specifically focused on the health and wellness industry. We connect health product and service brands with trusted influencers, and help influencers' followers access relevant and reliable products and services. The main problem we solve is that brands face challenges in finding suitable, genuinely engaging, and measurable influencers, while influencers lack the tools for professional collaboration and monetization. Our platform will create transparency, trustworthiness, and efficiency in influencer marketing within the health and wellness sector.

<b>2. Business & User Niche</b>
•   <b>Brands (B2B):</b> Companies producing health foods, fitness equipment, supplements, skincare products, and providers of health and wellness services, seeking targeted marketing channels and authentic consumer reach.
    •   Pain Points: Difficulty finding relevant target influencers, challenging to verify authenticity and effectiveness, managing multiple campaigns simultaneously, measuring Return on Investment (ROI), and high agency fees.
•   <b>Influencers (B2C for our platform, B2B for brands):</b> Fitness trainers, nutritionists, yoga instructors, mental wellness coaches, health advocates with existing followers.
    •   Pain Points: Lack of direct connections with quality brands, inconsistent income, difficulty managing multiple collaborations, content creation, and professional reporting.
•   <b>Followers/Customers (B2C - end-users):</b> Individuals interested in health, fitness, nutrition, mental wellness, seeking trusted product/service recommendations from experts.

<b>3. Revenue Model</b>
•   <b>Commission-based:</b> A percentage fee collected from the value of successful brand-influencer campaigns facilitated through the platform.
•   <b>Subscription Tiers for Brands:</b> Offering premium features (e.g., advanced influencer insights, detailed campaign analytics), priority influencer matching, and dedicated support.
•   <b>Subscription Tiers for Influencers:</b> Enhanced profile visibility, access to advanced insights on audience and performance, professional tools for campaign management, early access to brand campaigns.
•   <b>Sponsored Content Tools/Templates:</b> Providing premium tools for creating high-quality, brand-compliant content.
•   <b>Affiliate Marketing Integration:</b> Option for influencers to earn commission on sales generated directly through their unique links on the platform (brand-managed offers).

<b>4. Platform & Tech Stack</b>
•   Frontend: React.js (with Next.js for Server-Side Rendering (SSR) and improved SEO)
•   Backend: Node.js (NestJS Framework for scalable and efficient architecture)
•   Database: PostgreSQL (for complex relational data, user accounts, campaigns, and reliable transaction management) and Redis (for Caching and session management for fast performance)
•   Infrastructure: AWS (Amazon Web Services for hosting (EC2), file storage (S3), databases (RDS), serverless functions (Lambda)), Cloudflare (for CDN and security), Docker & Kubernetes (for scalability and deployment management)
•   Third-party APIs: Omise/Stripe (Payment Gateway for secure transactions), Instagram API, TikTok API, YouTube Data API, Facebook Graph API (for fetching influencer data like follower count and engagement rates), SendGrid (Email Service for communications)
